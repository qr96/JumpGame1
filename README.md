## 2. 설치 방법
  1. 파일 다운
    실행파일.zip 파일을 다운받아 줍니다.
  2. 압축 해제
    실행파일.zip 파일을 압축해제 해줍니다.
  3. 파일 실행
    압축 해제된 폴더 내부에 "Test1.exe"라는 파일을 실행해줍니다.
